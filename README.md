# PhonePe Transaction Analysis

## Project Description
This project analyzes PhonePe digital payment transactions to understand user behavior, transaction trends, and regional usage patterns. The system uses data visualization and analysis techniques to derive insights from transaction data.

## Objectives
- Analyze transaction trends over time
- Identify popular payment categories
- Compare state-wise and year-wise usage
- Generate actionable insights for fintech analysis

## Technologies Used
- Python
- Pandas & NumPy
- Matplotlib / Seaborn / Plotly
- Jupyter Notebook / Streamlit

## Applications
- Digital payment trend analysis
- Fintech business insights
- User behavior study
